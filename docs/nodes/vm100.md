# VM100 Services (GPU / Media)

## Purpose

- GPU-accelerated media streaming (Jellyfin with NVIDIA hardware transcoding)
- Audiobook hosting (Audiobookshelf)
- Primary media consumption endpoint for LAN and Tailnet clients

This document describes the current containerized media services running on VM100 (Compute/GPU VM).

## Jellyfin

- Deployment: Docker Compose (`/opt/homelab-server-architecture/docker/jellyfin/docker-compose.yml` on VM100)
- GPU acceleration: enabled (`gpus: all` + NVIDIA environment variables)
- Runs as non-root user (`user: 1000:1000`)
- Persistent application data:
  - `${JF_CONFIG}` Ôćĺ `/config`
  - `${JF_CACHE}` Ôćĺ `/cache`
  - `${JF_METADATA}` Ôćĺ `/metadata`
- Media mounts (read-only, least privilege):
  - `${JF_MEDIA_FILME}` Ôćĺ `/media/Filme:ro`
  - `${JF_MEDIA_SERIEN}` Ôćĺ `/media/Serien:ro`
- Port exposure: `8096/TCP` (LAN)

## Audiobookshelf

- Deployment: Docker Compose (`/opt/homelab-server-architecture/docker/audiobookshelf/docker-compose.yml` on VM100)
- Media mounts (read-only, least privilege):
  - `${ABS_AUDIOBOOKS_ROOT}` Ôćĺ `/audiobooks:ro`
  - `${ABS_PODCASTS_ROOT}` Ôćĺ `/podcasts:ro`
- Port exposure: `13378/TCP` (LAN)

## Storage Mount Strategy (Reboot-Safe)

Media paths on VM100 are provided via **systemd automount** units (fstype `autofs`) to ensure reboot-safe startup behavior:
- Jellyfin media path (referenced by `${JF_MEDIA_*}` variables)
- Audiobookshelf media path (referenced by `${ABS_*_ROOT}` variables)

Note: A legacy `media-mergerfs.mount` unit exists on VM100 but is not used for service mounts in the current state.

## Network / Tailscale
- Tailscale hostname: gpu-vm
- Tag: `tag:tier2`
- Accessible by: `tag:admin` (all ports), `tag:tier0` (all ports), `tag:tier2` (all ports), `tag:client` (ports 443, 8096, 13378), `tag:untrusted` (ports 443, 8096, 13378)

### Tailscale Serve

Not in use. Jellyfin and Audiobookshelf bind to `0.0.0.0` for LAN streaming performance.
Remote access uses the Tailscale IP directly (WireGuard-encrypted, no TLS hostname).

See: [Loopback + Tailscale Serve ADR](../decisions/loopback-tailscale-serve.md) ÔÇö section "Documented Exceptions"

## Security / Exposure Notes (Trade-off)

- No public reverse proxy and no router port-forwarding.
- Default remote access path is Tailscale (identity-based overlay).
- Network policy enforcement is defined via Tailscale ACL (tags + JSON policy).
  See: [docs/platform/tailscale-acl.md](../platform/tailscale-acl.md)

### Intentional LAN Exposure

- Jellyfin (`8096/TCP`) and Audiobookshelf (`13378/TCP`) are intentionally reachable on the LAN.
- This is a documented performance trade-off (bandwidth/latency optimization and reduced transcoding load).
- LAN exposure must remain limited to these specific ports only.
